# 🌍 AI Translator Website

A modern AI-powered multilingual translation web application built using HTML, CSS, and JavaScript.

## 🚀 Features

- Real-time language translation
- Hinglish support
- Voice input 🎤
- Text-to-speech 🔊
- Translation history
- AI typing animation
- Character counter
- Responsive glassmorphism UI
- Modern animations

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- MyMemory Translation API
- Web Speech API
